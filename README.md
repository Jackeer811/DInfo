# DInfo

```
Instalación:

Instalar la app termux-api en la play store

- En la terminal
$ apt update && apt upgrade
$ pkg install -y git
$ git clone https://github.com/Jackeer811/DInfo
$ cd DInfo
$ chmod +x Script.sh
$ ./Script.sh
```
